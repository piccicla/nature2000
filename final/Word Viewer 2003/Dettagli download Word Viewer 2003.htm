Panoramica

Il Word Viewer 2003 consente di aprire i documenti Word 2003 e quelli creati con tutte le versioni precedenti di Microsoft Word per Windows® e Microsoft Word per Macintosh. Oltre ai file di documenti Word (.doc) è possibile aprire file salvati nei seguenti formati:

  • Rich Text Format (.rtf)
  • Testo (.txt)
  • Pagine Web (.htm, .html, .mht, .mhtml)
  • WordPerfect 5.x (.wpd)
  • WordPerfect 6.x (.doc, .wpd)
  • Works 6.0 (.wps)
  • Works 7.0 (.wps)
  • XML (.xml)

Il Word Viewer 2003 consente di visualizzare, stampare e copiare il contenuto di documenti in un'altra applicazione. Non è tuttavia possibile modificare un documento aperto, salvare un documento o crearne uno nuovo. Questo download sostituisce il Word Viewer 97 e le versioni precedenti del Word Viewer.

Istruzioni

Installazione del download
  1. Fare clic sul pulsante Download (sopra) per scaricare il file e salvarlo sul disco rigido.
  2. Fare doppio clic sul file wdviewer.exe sul disco rigido per avviare il programma di installazione.
  3. Seguire le istruzioni visualizzate per completare l’installazione.
Istruzioni per l’uso

Al termine dell'installazione, aprire un file con il Word Viewer 2003:
  1. Scegliere Apri dal menu File.
  2. Nell'elenco Cerca in, selezionare l'unità, la cartella o l'URL che contiene il file da aprire.
  3. Nell'elenco di cartelle, individuare il file oppure aprire la cartella che contiene il file.
  4. Fare clic sul file e scegliere Apri.
Nota: Per esaminare il contenuto del file prima di scegliere Apri, selezionare Anteprima dal menu Visualizzazioni.

Rimozione del download

Per rimuovere il file di download, eliminare il file wdviewer.exe.

  1. Scegliere Impostazioni e Pannello di controllo dal menu Start.
  2. Fare doppio clic su Installazione applicazioni.
  3. Nell'elenco dei programmi installati, selezionare Microsoft Office Word Viewer 2003, quindi scegliere Rimuovi oppure Aggiungi/Rimuovi. Se viene visualizzata una finestra di dialogo, seguire le istruzioni per rimuovere il programma.
  4. Scegliere Sì oppure OK per confermare la rimozione del programma.
