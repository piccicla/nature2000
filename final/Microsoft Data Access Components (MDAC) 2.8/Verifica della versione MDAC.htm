Verifica della versione MDAC

Sommario

In questo articolo sono descritti due diversi metodi per verificare quale versione di Microsoft Data Access Components (MDAC) è installata in un sistema:
• Utilizzo dello strumento Component Checker.
• Verifica delle informazioni sulla versione memorizzate nel Registro di sistema.

Torna all'inizio

Installazione e utilizzo dello strumento Component Checker

Il metodo più affidabile per determinare quale versione di MDAC è stata installata consiste nel confrontare il numero di versione di ogni file DLL di MDAC con un elenco dei file DLL presenti in ogni versione di MDAC. Lo strumento Component Checker consente di effettuare questa operazione controllando i file presenti nel computer, confrontandoli con un elenco relativo a ogni versione di MDAC e riportando la corrispondenza più esatta.

Per installare Component Checker, attenersi alla seguente procedura:
1. Visitare il seguente sito Web Microsoft (informazioni in lingua inglese):
http://www.microsoft.com/downloads/details.aspx?FamilyId=8F0A8DF6-4A21-4B43-BF53-14332EF092C9&displaylang=en (http://www.microsoft.com/downloads/details.aspx?FamilyId=8F0A8DF6-4A21-4B43-BF53-14332EF092C9&displaylang=en)
2. Fare clic sul collegamento per effettuare il download di Component Checker. Quando richiesto, salvare il file Cc.exe (un file eseguibile autoestraente) sul desktop.
3. Sul desktop fare doppio clic sul fileCc.exe. In questo modo i file di Component Checker verranno estratti e installati nel percorso predefinito, C:\Comcheck.
Per utilizzare Component Checker e verificare la versione di MDAC in uso, attenersi alla seguente procedura:
1. Fare clic sul pulsante Start e scegliere Esegui.
2. Nella casella Apri digitare c:\comcheck\comcheck.exe , quindi scegliere OK.
3. Nella finestra di dialogo relativa alla scelta del tipo di analisi selezionare l'opzione relativa all'esecuzione dell'analisi del computer e alla determinazione automatica della versione quindi scegliere OK.
4. Verrà effettuato il tentativo di identificare la versione di MDAC installata nel computer mediante la scansione di tutti i file MDAC di base e delle impostazioni del Registro di sistema. Sono necessari alcuni minuti per il completamento di questa operazione. Una volta terminata la scansione, verrà visualizzato un messaggio analogo al seguente: 
La versione di MDAC corrispondente a quella installata nel computer è "XXXX".
5. Scegliere OK.
6. Verrà visualizzato un riepilogo dei file e delle impostazioni inclusi nella scansione. È possibile ignorare gli errori Dir, FileDescription e FileSize.

Controllo delle informazioni sulla versione memorizzate nel Registro di sistema

Sebbene non rappresenti il metodo più sicuro per controllare la versione di MDAC installata, il controllo delle informazioni sulla versione memorizzate nel Registro di sistema è un metodo semplice (nel caso in cui non si siano verificati problemi relativi a MDAC).

Le informazioni sulla versione sono contenute nella seguente chiave del Registro di sistema:
HKEY_LOCAL_MACHINE\Software\Microsoft\DataAccess\FullInstallVer
Per controllare il Registro di sistema, attenersi alla seguente procedura:
1. Fare clic sul pulsante Start e scegliere Esegui.
2. Nella casella Apri digitare regedit, quindi scegliere OK per avviare l'editor del Registro di sistema.
3. Nel riquadro di spostamento passare alla seguente chiave del Registro di sistema:
HKEY_LOCAL_MACHINE\Software\Microsoft\DataAccess
4. Nel riquadro dei dettagli individuare nella colonna NomeFullInstallVer e Version. Per ognuna di queste chiavi del Registro di sistema esistono informazioni sulla versione corrispondenti nella colonna Dati.
5. Al termine scegliere Esci dal menu Registro di sistema per chiudere l'editor del Registro di sistema.

Risoluzione dei problemi

Le informazioni sulla versione memorizzate nel Registro di sistema potrebbero non essere corrette in relazione alle versioni di MDAC precedenti alla 2.1 se confrontate con le versioni dei file correnti. Con Windows 2000 viene installata la versione 2.5. In Windows 2000 è possibile installare solo le versioni di MDAC successive alla 2.5.

I download relativi a Microsoft Data Access Components sono disponibili sul seguente sito Web (informazioni in lingua inglese):
http://msdn.microsoft.com/library/default.asp?url=/downloads/list/dataaccess.asp (http://msdn.microsoft.com/library/default.asp?url=/downloads/list/dataaccess.asp)
