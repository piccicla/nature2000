Panoramica


Insieme a Microsoft .NET Framework versione 2.0 Redistributable Package viene installato il runtime .NET Framework e i file associati necessari per eseguire le applicazioni sviluppate per .NET Framework 2.0.

Con .NET Framework versione 2.0 risultano migliorati la scalabilità e le prestazioni delle applicazioni in relazione alla memorizzazione nella cache, allo sviluppo e all'aggiornamento delle applicazioni tramite ClickOnce, il supporto per la più ampia gamma di browser e dispositivi con controlli e servizi ASP.NET 2.0. Per ulteriori informazioni su .NET Framework 2.0, fare clic qui (informazioni in lingua inglese).

Importante: non è possibile installare .NET Framework in due differenti lingue nella stessa macchina. Se si tenta di installare una seconda versione di .NET Framework in un'altra lingua, verrà visualizzato il seguente messaggio di errore: "Impossibile installare Microsoft .NET Framework perché è già installata un'altra versione del prodotto." se la piattaforma in uso non è nella lingua inglese o se si desidera visualizzare le risorse .NET Framework in una lingua differente, è necessario scaricare .NET Framework Language Pack nella lingua appropriata.

Requisiti di sistema

  • Sistemi operativi supportati: Windows 2000 Service Pack 3; Windows 98; Windows 98 Second Edition; Windows ME; Windows Server 2003; Windows Vista Business; Windows Vista Business 64-bit edition; Windows Vista Enterprise; Windows Vista Enterprise 64-bit edition; Windows Vista Home Basic; Windows Vista Home Basic 64-bit edition; Windows Vista Home Premium; Windows Vista Home Premium 64-bit edition; Windows Vista Starter; Windows Vista Ultimate; Windows Vista Ultimate 64-bit edition; Windows XP Service Pack 2
  • Software necessario:
    • Windows Installer 3.0 (ad eccezione di Windows 98/ME che richiede Windows Installer 2.0 o versione successiva). Windows Installer 3.1 o versione successiva (consigliato).
    • IE 5.01 o versione successiva: per installare .NET Framework è necessario che sia installato anche Microsoft Internet Explorer 5.01 o versione successiva.
  • Requisiti di spazio su disco: 280 MB (x86), 610 MB (x64)

Istruzioni

  1. Importante: accertarsi che siano installati il Service Pack e gli aggiornamenti più recenti della versione di Windows in esecuzione. Gli aggiornamenti più recenti sulla protezione sono disponibili sul sito Web Windows Update.
  2. Per iniziare a scaricare, fare clic sul pulsante Download nell'angolo superiore destro della pagina.
  3. Effettuare una delle seguenti operazioni.
    • Per iniziare immediatamente l'installazione, fare clic su Esegui.
    • Per salvare il file scaricato nel computer ed effettuare l'installazione in un secondo momento, fare clic su Salva.
    • Per annullare l'installazione, fare clic su Annulla.
    • Dopo aver installato .NET Framework, è necessario anche installare .NET Framework Language Pack nella propria lingua per completare la configurazione dell'ambiente .NET Framework. È necessario scaricare e installare .NET Framework Language Pack separatamente.

IMPORTANTE: se sono state installate le versioni preliminari di .NET Framework 2.0, ad esempio la build Beta 1, Beta 2 o Community Technical Preview (CTP), è necessario disinstallare tali versioni utilizzando Installazione applicazioni nel Pannello di controllo prima di installare la versione finale.

Informazioni aggiuntive

Requisiti per l'installazione del server

Per effettuare l'installazione di un server, è necessario che oltre ai normali requisiti di installazione, siano installati anche i seguenti programmi:

• Microsoft Data Access Components 2.8 o versione successiva (consigliato).
• Internet Information Services (IIS) 5.0 o versione successiva. Per accedere alle funzionalità di ASP.NET, prima di installare .NET Framework è necessario installare IIS con gli ultimi aggiornamenti sulla protezione. ASP.NET è supportato solo sulle seguenti piattaforme: Microsoft Windows 2000 Professional (Service Pack 3 consigliato), Microsoft Windows 2000 Server (Service Pack 3 consigliato), Microsoft Windows XP Professional e la famiglia Microsoft Windows Server 2003.

Nota: con l'installazione tipica è possibile ignorare tali requisiti. Se non si è certi del tipo di installazione da eseguire, soddisfare unicamente i requisiti dell'installazione tipica.

Per lo sviluppo delle applicazioni .NET Framework versione 2.0, Microsoft .NET Framework versione 2.0 Software Development Kit (SDK) include gli strumenti, la documentazione e gli esempi necessari per scrivere, generare, eseguire i test e installare le applicazioni .NET Framework.
